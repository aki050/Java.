
# The Algorithms - Java（日本語）

このリポジトリは、さまざまなアルゴリズムをJavaで実装したものです。教育目的で作成されており、標準ライブラリより効率が劣る場合もありますが、学習や理解の助けとなることを目指しています。

---

## プロジェクト概要

- すべてのアルゴリズムはJavaで実装されています。
- 各アルゴリズムは独立したクラスとして整理されています。
- [ディレクトリ一覧](DIRECTORY.md)から全アルゴリズムを確認できます。

## 貢献方法

貢献の前に、必ず[貢献ガイドライン](CONTRIBUTING.md)をお読みください。

### バグ報告
- 既知のバグかどうか[Issue](https://github.com/TheAlgorithms/Java/issues)で検索してください。
- 新しいバグの場合は、タイトル・詳細・再現手順を明記してIssueを作成してください。

### ドキュメントや新機能の追加
- リポジトリをForkし、変更を加えてPull Requestを作成してください。
- 新機能の場合は、タイトル・詳細・テストケースをIssueに記載してください。

### 質問
- [Gitter](https://gitter.im/TheAlgorithms/community?source=orgpage#)やIssueで質問できます。

## ディレクトリ構成

- 詳細は [DIRECTORY.md](DIRECTORY.md) をご覧ください。
